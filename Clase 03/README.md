# Clase 03 - GIT

## Repaso de Branch

### Listo ramas
```sh
git branch
```

### Creo una rama
```sh
git branch <nombre-rama>
```

### Cambio entre ramas
```sh
git switch <nombre-rama>
```

## Git Merge

### Tipos de Merge
